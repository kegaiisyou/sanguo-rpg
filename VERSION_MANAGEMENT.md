## 🚀 部署指南

### 使用一键部署+备份（新版本推荐）

1. 双击运行 `一键部署+备份.bat`
2. 输入新版本号（如 v1.4.0）
3. 输入更新内容描述
4. 等待自动推送到 GitHub

### 使用一键部署（快速更新）

1. 双击运行 `一键部署.bat`
2. 直接回车跳过说明
3. 等待推送完成

## 🛠️ 开发流程

### 更新前检查清单

- [ ] 阅读 `版本管理手册.md` 了解完整流程
- [ ] 确认 `index.html` 是最新修改的
- [ ] 确保 `backup/` 文件夹存在

### 更新步骤

1. **修改代码** → 编辑 `index.html`
2. **运行部署脚本** → 双击 `一键部署+备份.bat`
3. **测试验证** → 手机访问游戏地址
4. **确认成功** → 命令行显示"部署完成"

## ⚠️ 注意事项

1. **不要删除 `backup/` 文件夹** - 包含所有历史版本
2. **每次更新都要运行部署脚本** - 自动备份+上传
3. **版本号要递增** - 方便追踪和管理
4. **手机测试后再算完成** - 确保 GitHub Pages 已更新
