# 三国江湖 RPG - 美术规范

## 一、整体风格定位

### 选定风格：东方古典像素风
- **参考游戏**：吞食天地、勇者斗恶龙（DQ）、星露谷物语
- **风格特点**：
  - 简洁的像素感角色
  - 古风水墨配色
  - 16x16 或 32x32 的网格系统
  - 清晰的轮廓和色块

---

## 二、色彩规范

### 主色调
| 名称 | 色值 | 用途 |
|------|------|------|
| 金色 | `#FFD700` | 玩家、高亮、重要元素 |
| 青绿 | `#2D5A2D` | 草地、树木 |
| 棕褐 | `#5A4A3A` | 建筑、木材 |
| 深蓝 | `#1A1A2E` | 夜空、深色背景 |
| 白色 | `#FFFFFF` | NPC轮廓、文字 |

| 角色 | 主色 | 辅色 | 轮廓 |

| 玩家 | 金色 `#FFD700` | 深黄 `#B8860B` | 白色 |

### UI配色
| 元素 | 颜色 |
|------|------|

| 背景 | 半透明黑 `rgba(0,0,0,0.85)` |

## 三、角色尺寸规范

### 基础网格
- **角色**：32x32 像素（显示时约 30-40px）
- **NPC**：32x32 像素
- **敌人**：32x32 像素
- **建筑**：64x48 或 48x48 像素
- **树木**：32x48 像素

### 角色结构
```
┌─────────┐
│   头    │  头顶（头发/帽子）
├─────────┤
│   身    │  身体（服装主色）
├─────────┤
│   脚    │  双脚/武器
└─────────┘
```

## 四、地图元素规范

### 草地
- 颜色：`#2D5A2D`（主色）
- 纹理：随机分布的小草点 `#3D6A3D`
- 格子感：16px 间隔的网格线

### 道路
- 颜色：`#8B7355`（泥土色）
- 可选：`#A08060`（石板路）

### 建筑
- 屋顶：`#8B4513`（深棕）
- 墙壁：`#DEB887`（浅棕）
- 门：`#654321`（深棕）
- 窗户：蓝色 `#6B8E9F`

### 树木
- 树冠：圆形，深绿 `#1A3A1A`
- 树干：棕色 `#5A4A3A`
- 尺寸：宽 32px，高 48px

### 水/井
- 颜色：深蓝 `#4A6A8A`
- 波纹效果：白色点缀

---

## 五、UI界面规范

### 状态栏
- 位置：右上角
- 背景：半透明黑色 `rgba(0,0,0,0.85)`
- 边框：金色 1px
- 圆角：8px
- 内边距：15px

### 对话框
- 位置：屏幕底部
- 高度：28%（可调整）
- 背景：深色半透明
- 顶部边框：金色 3px

### 触控按钮
- 形状：圆形
- 大小：60-70px
- 背景：半透明 + 边框
- 图标：emoji 或简单图形

## 八、实现方式

由于当前使用纯 CSS/HTML 实现，将采用：
1. **CSS 伪元素**绘制角色
2. **CSS 渐变**创建纹理
3. **SVG 内联**绘制复杂图形
4. **Emoji** 作为图标补充
