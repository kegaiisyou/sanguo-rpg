## 一、世界观（四层 + 部曲/豪商/异族）

### 1.1 时间设定
天下表面承平，暗流汹涌：太平道民间渗透、十常侍弄权、西凉边军坐大、党锢名士蛰伏。主角在"暴风雨前夜"起步，随声望/事件被卷入即将到来的大乱。**佛教初传、尚弱**（白马寺仅为洛阳小僧团，异质支线）；**童渊/百鸟朝凤枪属评书传说层**，需标注。

- **东汉朝廷·洛阳**：灵帝 + 十常侍 + 外戚（何进胚）。
- **洛阳禁军**：虎贲营 / 羽林营 → **王越剑术一脉**（京城武学源流，师徒+武馆制，正史《典论》）。

## 三、资源系统（三轨）

| 资源 | 类别 | 说明 |
|------|------|------|
| 生命 | 生存 | 原"气血"。归 0 → **死亡** → 回标题页（读档/重开） |
| 体力 | 生存（行动点） | 挖矿/赶路/复杂交互/部分战斗动作消耗；**休息**恢复。**亏空=无法行动**，但不直接掉血 |
| 饥饿值 | 生存（饥饿） | 合并原"食/水"的生存指标，随时辰/行动缓慢上升（越来越饿）；**饥饿值过高（饿极）= 持续扣生命**。**饮食降低饥饿值**（与体力分开结算，体力靠休息恢复） |
| 银两 | 经济（货币） | 顶层货币计数（买卖/悬赏/投资）；特殊形态（银票/金锭/借据/假银）作为背包物品具象化，用于藏匿、大额交易、骗术事件 |
| 潜能 | 修习 | 修炼武学消耗（**稀缺资源**，设周获取上限，学武需取舍） |
| 内力 | 内修（门控） | **默认隐藏**，学"内功/心法"后现身并启用；内力运用有独立熟练度 |
| 声望 | 知名度 | 0–100，仅影响知名度/称号/解锁（不进战力） |
| 侠义值 / 凶名值 | 道德（双轨） | 各自独立计，影响 NPC 态度与内容解锁 |

> **饥饿值 与 体力 不重复**：饥饿值管"生死"（饿极掉血，靠饮食降低），体力管"能否行动"（累得挖不动，靠休息恢复）；机制目的不同——可独立出现"精力充沛却饿极"或"吃饱却累得挖不动"两种受限状态。

**死亡与存档**：死亡回标题；`localStorage` 自动存档（记录时间戳）；"更多"内可手动存档/清档。

## 四、成长系统（多轨等级）

### 4.0 先天资质（角色创建属性，不计入战力，影响成长底色）
角色出生时按出身/资质分配，可通过开窍丹/奇遇微调：
- **悟性**：影响秘籍学习门槛与熟练度获取速率（见 4.3.1）。
- **根骨**：影响气血上限与防御成长（硬功/肉盾底色）。
- **身法**：影响速度/闪避成长（轻灵/先手底色）。
- **福缘**：影响奇遇触发率与掉落品质（机缘底色）。
- **功法亲和**：按系（枪/拳/剑/刀…）提速该系武学的学习与突破（见 4.3.5）。

### 4.1 声望等级（0–100，称号区间，不频繁换）
| 声望 | 称号 |
|------|------|
| 0–9 | 无名小卒 |
| 10–24 | 初入江湖 |
| 25–39 | 小有名气 |
| 40–54 | 江湖新秀 |
| 55–69 | 名动一方 |
| 70–84 | 威震一方 |
| 85–94 | 名扬天下 |

### 4.2 善恶双轨（侠义值 / 凶名值，各自独立）
- 两值互不抵消，分别累积，各自解锁内容：
  - `侠义 ≥ 30` → 清流名士敬重、结交名士、接侠义委托。
  - `凶名 ≥ 30` → 影门/匪类亲近、接高阶悬赏、威吓类对话。
  - `侠义≥40 ∧ 凶名≥40` → 「亦正亦邪·枭雄」隐藏线 + 专属称号/事件。
- NPC 态度由双轴共同决定（敬重/友善/平常/戒备/敌视）。

### 4.3 武器艺技能线（13 线，独立升级，实战+修炼双成长）
每条线对应武器实战/修炼涨熟练度，影响该武器战力：

| 技能线 | 涵盖武器 | 分支/流派 | 主影响属性 |
|--------|----------|-----------|------------|
| 拳脚 | 拳、掌、腿（徒手） | 拳法/掌法/腿法（高拳脚解锁分支） | 攻(徒手)/暴击/内力附着基底 |
| 剑法 | 剑、双股剑 | 双持剑（高剑法解锁） | 攻/命中 |
| 刀法 | 刀、双刀、陌刀、盾刀 | 双刀(双持)·重刀(陌刀)·刀盾(盾刀) | 攻/暴击/防(盾) |
| 枪法 | 枪、矛 | 矛=枪法变体 | 攻/命中 |
| 棍法 | 棍、三节棍 | 软棍(三节棍) | 攻/防 |
| 戟法 | 方天画戟 | 长兵奇门(吕布系) | 攻/范围 |
| 锤法 | 锤、骨朵等钝器 | 重兵 | 攻/破甲/眩晕 |
| 鞭法 | 鞭、软鞭链子 | 软兵 | 命中/减速 |
| 弓术 | 弓弩 | 远兵 | 攻(远)/命中 |
| 暗器 | 飞镖/毒针 | 影门/巴蜀 | 命中/速度 |
| 骑术 | 马战 | 西凉/白马义从 | 速度/攻(骑) |
| 轻功 | 身法 | — | 速度/闪避 |
| 内功 | 心法/内修 | **内力运用熟练度**（子项） | 解锁内力条+护体+运劲效率 |

**内力运用熟练度**：`内功等级`=修为深度（内力上限/回复/护体）；`内力运用熟练度`=子熟练度（技能耗内力折扣、持续类内功时长、内力技暴击），每次释放内力技涨。

### 4.3.1 武器艺线的解锁与晋升
- **实战积累**：战斗中按出手类型累积对应技能线经验——出拳→「拳法」、出掌→「掌法」、出腿→「腿法」；持枪→「枪法」、使刀→「刀法」……经验达标自动升级，提升该线战力加成。
- **门派传功**：完成门派任务 → 传功长老教学 → 直接授予招式（门派福利，免去秘籍）。
- **秘籍研读**：特殊 Boss / 奇遇掉落「武功秘籍」物品 → 入背包 → 研读习得其中招式。
- **学习门槛**：习得某招式需满足 `悟性 ≥ X` 且 `对应技能线等级 ≥ Y`（部分高阶招式另需 `内功等级 ≥ Z`）；不满足时秘籍可读但招式"暂不可悟"。
- **资质·悟性**：角色固有资质（选角/出生定基础），影响秘籍学习门槛与熟练度获取速率；可通过开窍丹/奇遇提升。记为独立属性 `悟性`，**不计入战力**。
- **潜能稀缺**：潜能设**周获取上限**（避免无限习得），学武需取舍——"精几门"而非"全都会"，塑造角色个性与 build 差异。

### 4.3.2 武学境界（修习深度，参考鬼谷八荒）
与「武器艺线」是**两个独立轴**，避免混淆：
- **武器艺线等级**（4.3）：你的"功底"，决定**能学哪些招式**（门槛）。
- **武学境界**：某套/某招的"精通度"，决定**这招多强、解锁什么特效**。
- 修习境界七阶：`初窥门径 → 小有成 → 小有成就 → 已臻大成 → 出神入化 → 登峰造极 → 返璞归真`。
- 每破一境按该武学「突破表」解锁增益：小成加招式、大成加连段/特效、化境加**心法共鸣**、返璞归真**觉醒奥义**（脱招式也能用的终极发力）。
- 突破需 `熟练度 + 突破材料(丹/属性晶) + 悟性门槛`，非自动满。
- **与潜能稀缺咬合**：突破消耗潜能，潜能周上限→逼你"精几门"而非全学，天然抑制复杂度。

### 4.3.3 武学构成（一套多式，参考下一站江湖）
一套武学是容器，含三类：
- **武技**：主动小招（有节拍值、耗内力/体力）。
- **招式**：绝技/大招（需战意满或内力满、有境界门槛，见 5.1.2）。
- **技巧**：被动/发力方式，分三种绑定：
  - **心法依附型**：仅当装备对应心法时生效（如"需配玄冥心法"）。
  - **条件触发型**：满血/残血/处于某 Buff/特定节拍下触发。
  - **脱离型·发力方式**（寸劲/螺旋劲/震字诀等）：可脱离本武学，嵌到任意武学使用（见 4.3.3.1）。

#### 4.3.3.1 发力技巧层（通用发力方式）
脱离型技巧作为"发力方式"可装配给当前主武学，提供通用运劲（如寸劲增暴、螺旋劲破甲、震字诀反震）。
- 占槽位/耗内力 → 逼权衡（堆发力 vs 留槽给绝技）。
- 好处：老玩家钻研回报高，且避免"每门武功从头练发力"的重复感（与潜能稀缺、相生共鸣同为"减负"机制）。

### 4.3.4 学招来源（多渠道，参考大侠立志传）
招式获取渠道（每个都简单，丰富但不复杂）：
1. **实战积累**：艺线升级自动解锁基础招式（4.3.1）。
2. **门派传功**：完成门派任务→长老授招（免去秘籍）。
3. **秘籍研读**：Boss/奇遇掉秘籍→研读习得（需悟性+艺线门槛）。
4. **切磋胜后**：比武胜出→对方拜服授招。
5. **偷师**：潜行观摩敌出招→领悟（福缘影响成功率）。
6. **残卷合成**：招式残页散落（接典籍争夺/影门）→集齐合成秘籍。

### 4.3.5 临阵顿悟与深层词条（第3层深水，可选）
- **临阵顿悟**：战斗中按 `福缘` 概率触发，临时领悟/强化某招（福缘驱动的战斗奇遇）。
- **随机词条**：每武学/秘籍带 1–3 条随机词条（如"崩拳·破甲+15%"），可借开窍/逆天改命事件替换→同招 build 差异。
- **功法亲和**：先天资质之一（见 4.0），按系（枪/拳/剑…）提速该系学习/突破。
- 以上均为可选深水区，新手不碰亦能通关。

### 4.4 属性模型与加成来源
`实战属性 = 基础值(体质/装备) + 当前所用武器对应技能线等级加成 + Buff/Debuff`
- **攻**：基础 + 当前武器线加成
- **防**：基础(装备/内功护体)
- **速度**：基础 + 轻功等级（决定先手 + 同回合可行动次数，由招式节拍与速度共同决定，见 5.1.1）
- **命中**：基础 + 当前武器线加成 − 目标闪避
- **闪避**：基础 + 轻功加成（封顶，避免无敌）
- **暴击**：基础 + 拳脚/特定功法加成（暴伤默认 1.5–2.0）
- 可延展接口（先留，P 级再开）：韧性/抗暴、招架/格挡、破甲。

### 4.5 内力与内功深层机制
- **基础内力附着攻击**：仅修内功、未习任何武学时，普攻可"附着内力"——消耗内力并获**微弱增伤**；但因无武学运劲方式，增伤极差，且单次消耗受内力上限**严格限制**（防止空有内力乱杀）。
- **每门武学独立内力运用方式**：不同武学耗内力/增伤/特效各异（如枪法走螺旋透劲、刀法走劈砸、剑法走轻灵），习得后替代/强化基础附着。
- **每门内功生成 / 修炼方式不同（场景依赖）**：
  - 需在**特定练功房**打坐吐纳；
  - 或于**水中游弋**修炼（水属内功）；
  - 或于**特殊场景**（雪山/丹室/阵法）修炼。
- **修炼场景判定**：每门内功带 `requiredScene`（如 `['练功房']` / `['水']` / `['雪山','丹室']`）。玩家在房间选"运转内功"时，判定当前房间 `tags` 是否含所需场景：
  - **匹配** → 正常积累内功经验 + 效率加成；
  - **不匹配** → 修炼效率极低，或（刚猛/阴寒功法）产生内伤/郁气，甚至走火入魔风险。
- **练功房来源**：属特定势力/门派/组织（少林藏经阁、武当静室、影门密窖），或剧情任务解锁的私密练功房；部分野外场景天然带 `水`/`雪山`/`丹室`/`阵法` 标签。
- **境界突破**：内功修至"大通任督二脉"后可**自行运转**，脱离场景依赖、内力自动回复，并解锁更高运劲效率。

### 4.6 功法属性与相生相克（两层属性模型）
属性分两层，各司其职（主网仅 7 值，新手记这层即可；异术属为稀有调味）：
- **内层·五行 + 阴阳**：
  - `金木水火土` 构成相生相克主网（火克金、水克火…），驱动相生共鸣。
  - `阴阳(刚柔)` 为垂直轴——**刚猛**取`刚`（破柔防、高攻），**阴柔**取`柔`（对刚减伤+反弹，"以柔克刚"）；管风格克制，不进五行网。
- **外层·异术属（稀有，不进相生网）**：留给概括不了五行/阴阳的特殊功法，每个只带一条签名机制（复用 5.3 的 Buff/Debuff，不增新规则）：

| 异术属 | 签名机制 | 典型出处 |
|---|---|---|
| 雷 | 破邪/麻痹(眩晕)、克阴邪 | 雷法、方士神怪层 |
| 毒 | 持续伤(中毒) | 巴蜀暗器、五斗米道 |
| 蛊 | 寄生/失控(混乱) | 南蛮蛊毒手 |
| 寒/冰 | 减速+封内力(郁气) | 雪山内功、水属变体 |
| 风 | 先手+范围 | 轻功奇门 |
| 幻 | 命中降/混乱 | 方士遁术 |
| 血 | 吸血/狂暴 | 凶名线、魔功 |
| 煞 | 压制低凶名敌(威吓) | 影门、修罗功 |

- **相生共鸣（套装特效）**：装备相生组合的内功+外功+身法（如"水内功+剑外功+轻功身法"）→触发隐藏共鸣特效（如"以柔克刚：受击转内力"）。鼓励搭配而非堆数值。

## 五、战斗系统

### 5.1 伤害公式
`伤害 = 攻 × 100 / (100 + 防)`（防御越高收益递减，避免堆死）。

### 5.1.1 出招耗时（招式节拍，替代 BUSY）
- 每招式带 **节拍值（出招耗时）**：施展后进入 **「运招中」** 状态，未竟前不可临时变招（除非特定"变招/抢占"类招式）。
- **速度 / 轻功** 降低自身节拍值：同回合内累计耗时未满阈值 → 触发**额外行动**（原"速度差触发额外行动"的底层来源，由此细化）。
- 重招（陌刀/锤/方天画戟）节拍长、单发威力高；轻招（拳脚/暗器/剑）节拍短、可连击——形成"刚猛 vs 灵巧"的节奏流派。
- 被控（眩晕/减速/郁气）会拉长效节奏拍，变相失去行动机会。

### 5.1.2 战意槽（独立于内力）
- 普攻/受击攒**战意**，满槽释放「招式(绝技)」（见 4.3.3）。
- 与内力形成"两套资源、两种玩法"：外功流靠战意爆发，内功流靠内力运劲，互不强制。
- 战意也可由特定招式/身法加速积累，作为 build 取向之一。

### 5.2 节奏：半手动
每回合玩家决策层：普攻 / 释放技能（耗内力或体力）/ 用道具 / 防御 / 逃跑。
- **自动战斗开关**：关闭时每回合等待玩家指令（半手动）；开启时由 AI 按"最优/保守"策略自动出招并加速演出。
- **文字演出节奏**（详见 5.4）：以战报流 + 可调速度档 + 关键招式逐字特写 + 跳字反馈呈现，无图形动画。
> 注：本作为纯文字 RPG，放置内核已移除；潜能来源统一为历练。

### 5.3 Buff / Debuff
- **增益(Buff)**：护盾·聚气·暴击增益·**兴奋**(攻/速↑)·**凝神**(命中/暴击↑)·**心流**(全属性微增、进入专注状态)。
- **减益(Debuff)**：流血·中毒·灼烧·眩晕·减速·**内伤**(内力上限/攻↓)·**郁气**(速度/内力回复↓)·**走火入魔**(内功不稳：内力紊乱、随机反噬，多由内力运用不当或境界未稳触发)。
- 由技能、武器分支、内功、道具、环境产生；走火入魔为内功专属危险态，需特定丹药/奇遇化解。

### 5.4 文字演出节奏（无图形动画）
- **速度档**：极速（瞬出全部战报）/ 普通（逐行淡入）/ 沉浸（逐字浮现 + 关键招式特写）。
- **跳字反馈**：伤害「-128」红字跳现、暴击「暴击!」金闪、闪避「闪避」灰显、内伤/走火入魔特殊警示。
- **出手可视化**：双方面板高亮当前行动者，速度条预示先手与额外行动。
- **连招演出**：多段攻击合并为招式描写（如"连环三式，枪出如龙"）。
- **自动战斗**开启时按所选策略跳过逐字、直接出结果战报。

### 5.5 敌人模板与掉落表
按敌人模板挂 weighted drop table（药材/兵器/防具/饰品/武学典籍/杂物/书信任务物品）。

### 5.6 Boss 战
高光时刻走半手动；华雄已在对话预留 defeat 文本，作为首场标志性 Boss。

## 八、影门（刺客组织）— 骨架 + 占位事件

## 十一、主线门控逻辑（世界状态机 + 声望 + 阵营认可）
主线节点 = `f(世界状态, 声望等级, 阵营认可度)`，可复用到所有章节。
示例·**斩华雄触发条件**：
```
诸侯会盟状态 == 进行中
∧ 华雄 == 存活
∧ 主角声望 ≥ 20
∧ 十八路诸侯认可度 ≥ 阈值
（主角阵营 ∈ 十八路诸侯  ∨  声望达标）
```
- 章节推进：剿黄巾 → 平西凉 → 斩华雄（后续章节随进度解锁）。
- 每阶段 = 等级/声望门槛 + 指定任务/Boss 战双重门控。
