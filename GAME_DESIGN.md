# 三国江湖 RPG - 游戏设计文档

## 一、游戏概述

**游戏名称**: 三国江湖  
**类型**: 回合制 RPG  
**平台**: 网页游戏 (HTML/CSS/JS)  
**时代背景**: 东汉末年，黄巾之乱  

---

## 二、核心设定

### 2.1 世界观
- **时间**: 公元184年（东汉光七年）
- **地点**: 荆州长沙郡
- **事件**: 黄巾起义爆发，天下大乱

### 2.2 主角设定
| 属性 | 值 |
|------|-----|
| 姓名 | 无名少年（可自定义） |
| 身份 | 武将后人，身世成谜 |
| 起始等级 | LV.1 |
| 起始属性 | HP:100, MP:50, ATK:10, DEF:5, SPD:5 |
| 金币 | 0 |

### 2.3 主角背景
```
你是一位身世成谜的青年，据说是某位武将的后人。
在一场神秘的变故后，你失去了记忆，醒来时发现自己躺在一片荒郊野外。
为了寻找真相和身世，你踏上了乱世的旅途...
```

---

## 三、剧情大纲

### 第一章：乱世开端
1. **开场**: 主角从昏迷中醒来，失去记忆
2. **长沙城**: 遇到士兵、将军，获得主线任务
3. **试炼**: 击败黄巾兵，证明实力
4. **第一个小BOSS**: 击败黄巾勇士

### 第二章：群雄并起（规划中）
- 前往洛阳，调查董卓之乱
- 结识更多三国名将

### 第三章：天下三分（规划中）
- 魏蜀吴三条主线分支
- 最终决战

## 四、游戏系统

### 4.1 角色属性
| 属性 | 说明 | 初始值 |
|------|------|--------|
| HP | 生命值 | 100 |
| MP | 法力值 | 50 |
| ATK | 攻击力 | 10 |
| DEF | 防御力 | 5 |
| SPD | 速度 | 5 |
| LV | 等级 | 1 |
| EXP | 经验值 | 0 |
| GOLD | 金币 | 0 |

### 4.2 战斗系统
- **回合制**: 玩家先手
- **行动选项**: 攻击 / 技能 / 物品 / 逃跑

#### 技能列表
| 技能名 | MP消耗 | 效果 |
|--------|--------|------|
| 普通攻击 | 0 | 造成100%攻击力伤害 |
| 重击 | 10 | 造成150%攻击力伤害 |
| 治愈术 | 15 | 恢复30%最大HP |

#### 物品列表
| 物品名 | 类型 | 效果 |
|--------|------|------|
| 草药 | 消耗品 | 恢复30HP |
| 法力药水 | 消耗品 | 恢复20MP |

### 4.3 升级系统
- 每升一级需要 100 经验
- 升级效果：
  - HP上限 +20
  - MP上限 +10
  - 攻击力 +3
  - 防御力 +2

### 4.4 任务系统
| 任务ID | 名称 | 描述 | 奖励 |
|--------|------|------|------|
| quest_001 | 征兵试炼 | 击败5名黄巾兵 | 200金币, 100经验 |
| quest_002 | 护卫商人 | 护送商人安全通过 | 300金币, 150经验 |

## 七、游戏流程（第一步）

### 7.1 标题界面
- 游戏标题 "三国江湖"
- "开始游戏" 按钮
- "继续游戏" 按钮（如果有存档）

### 7.2 开场剧情
```
说书人：
话说天下大势，分久必合，合久必分。
东汉末年，朝纲不振，宦官当道，民不聊生。
张角以"太平道"为号，发动黄巾之乱，天下震动。

而你，是一位身世成谜的青年，
在乱世的烽火中，踏上了寻找真相的道路...
```

### 7.3 地图
- 简单的横版地图
- 可移动的主角（金色方块）
- 3个可交互NPC

### 7.4 操作
| 按键 | 功能 |
|------|------|
| ← → | 移动 |
| 空格 | 与NPC对话/确认 |
| I | 背包 |
| K | 技能 |
| ESC | 关闭对话 |
