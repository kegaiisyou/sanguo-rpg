# 三国江湖 - 角色捏脸系统 (v1.6.0)

## 版本说明
v1.6.0 - 实现精细角色捏脸系统，所有角色（玩家/NPC/敌人）共享可复用的美术资源

## 技术实现

### SVG角色结构
```
<svg class="character">
  <g class="body">        <!-- 身体阴影 -->
  <g class="legs">       <!-- 腿部 -->
  <g class="cloth">      <!-- 服装下身 -->
  <g class="cloth-upper"> <!-- 服装上半身 -->
  <g class="weapon">     <!-- 武器 -->
  <g class="head">
    <g class="face">     <!-- 脸型 -->
    <g class="eyes">     <!-- 眼睛 -->
    <g class="eyebrows"> <!-- 眉毛 -->
    <g class="nose">     <!-- 鼻子 -->
    <g class="mouth">    <!-- 嘴巴 -->
    <g class="ears">     <!-- 耳朵 -->
  </g>
  <g class="hair">      <!-- 发型 -->
  <g class="hat">       <!-- 帽子/头饰 -->
</svg>
```

### 角色生成函数
```javascript
function createCharacter(config) {
  // config: { face, eye, brow, hair, hairColor, skin, cloth, clothColor, weapon }
  // 返回SVG元素
}
```

## 动画支持
- idle: 待机呼吸动画
- walk: 行走动画
- attack: 攻击动画
- hurt: 受伤动画
- death: 死亡动画
